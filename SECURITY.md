# Security policy

Report suspected vulnerabilities privately through GitHub Security Advisories for this repository.
Do not open a public issue or include live credentials, personal data, or exploit details in logs.
The maintainers will acknowledge a complete report and coordinate remediation and disclosure.

Only the latest released minor line is eligible for security fixes. This foundation is pre-1.0
and has not undergone an external security audit. Never deploy the development bootstrap key in
production.
